# ProScore-Public
ProScore is a lightweight Chrome extension that helps you find the best value on protein supplements. By analyzing both the actual protein percentage and the cost per gram of protein, ProScore instantly calculates a value metric and ranks products side-by-side so you never overpay for filler.
